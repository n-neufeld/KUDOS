Kudos, a peer-to-peer social media celebration platform to praise those you work with!

## Installation

Install my-project with npm

```zsh
  cd client
  npm install
```

```node
  cd server
  node install
```

## Run

```zsh
  cd server
  npm run dev
```

Run both server and client.

## Features

- Create a Kudo
- Comment on a Kudo
- Draw a unique canvas on create Kudo
- Responsive design
